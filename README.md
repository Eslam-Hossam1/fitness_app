## 📱 About IronPulse

**IronPulse** is a high-performance fitness app that connects athletes with professional workout plans and expert trainers. Browse curated training programs, save your favorites, track progress, and fuel your transformation — all from one sleek, dark-themed interface.

---

## ✨ Features

| Feature             | Description                                  |
| ------------------- | -------------------------------------------- |
| 🏠 **Home Feed**     | Discover featured workout plans by level     |
| 🗂️ **Plans Library** | Browse Beginner → Advanced training programs |
| 📋 **Plan Details**  | Full week-by-week workout breakdown          |
| 🏋️ **Trainers**      | Browse certified coaches by specialty        |
| ❤️ **Favorites**     | Save and organize your go-to routines        |
| 👤 **Profile**       | Manage account, stats, and settings          |
| ✏️ **Edit Profile**  | Update info & profile picture                |
| 🔐 **Auth Flow**     | Sign Up, Log In, Google Sign-In              |

## 🛠️ Tech Stack

```
IronPulse
├── UI Layer          → Flutter + flutter_screenutil + cached_network_image
├── State Management  → BLoC / Cubit + HydratedBloc
├── Architecture      → Clean Architecture (Data / Domain / Presentation)
├── Backend           → Supabase (Auth + Database + Realtime)
├── Networking        → Dio + REST API
├── DI                → GetIt
└── Local Storage     → Hive + Shared Preferences
```

## 🚀 Getting Started

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/ironpulse.git

# 2. Install dependencies
flutter pub get

# 3. Run the app
flutter run
```

> Make sure you have Flutter SDK installed and a connected device or emulator.
